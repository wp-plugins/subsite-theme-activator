Activate a theme across all subsites. For use with multisite networks only.

== Description ==

This plugin is designed to work with multisite networks where the same theme is used across all sites.
After network enabling a new theme, a new action is available called "Subsite Activate."
Clicking this action will enable the theme for every subsite. 

== Installation ==

1. Upload `subsite-theme-activator.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
